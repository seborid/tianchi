# 天池移动推荐算法竞赛-基线方案  

## 环境要求  
- **Python 版本**: 3.8+  

## 安装与运行  

1.  **克隆或下载项目**  

2.  **创建并激活虚拟环境** (推荐)  
    ```shell
    # 创建名为 venv 的虚拟环境
    python -m venv venv

    # 激活虚拟环境 (Windows)
    .\venv\Scripts\activate
    ```

3.  **安装依赖**
    项目所需的所有依赖库都记录在 `requirements.txt` 文件中。运行以下命令进行安装：
    ```shell
    pip install -r requirements.txt
    ```

4.  **运行代码**
    ```shell
    python main.py
    ```
